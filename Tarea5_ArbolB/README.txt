***** IMPORTANTE *****

El código, al ejecutarlo, muestra un menú interactivo que permite trabajar con un Árbol B configurable (puedes definir el grado mínimo del árbol al iniciar el programa). 
Desde el menú, puedes realizar varias operaciones:

- Insertar números en el árbol B.
- Buscar números dentro del árbol.
- Eliminar números del árbol, manteniendo siempre el equilibrio automáticamente.
- Cargar listas de números directamente desde archivos .CSV para insertarlos en el árbol.
- Generar y visualizar automáticamente un gráfico del Árbol B usando Graphviz. 
  (El gráfico también se guarda en la misma carpeta donde se encuentra el proyecto como un archivo .png).

El menú tiene la siguiente apariencia:

=== MENU ===
1. INSERTAR UN NÚMERO
2. BUSCAR UN NÚMERO
3. ELIMINAR UN NÚMERO
4. CARGAR LISTA DE NÚMEROS DESDE ARCHIVO .CSV
5. GENERAR GRÁFICO
6. SALIR

Nota: El árbol B se mantiene balanceado automáticamente luego de cada operación de inserción o eliminación, garantizando un rendimiento óptimo en todo momento.
